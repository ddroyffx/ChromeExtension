Conformité RGPD / UK-GDPR – Extension Acheter Durable

1. Données collectées

  • ✅ Ne collecte aucune information personnelle (nom, courriel, etc.).
  • ✅ N’utilise que le sessionStorage, stocké localement et temporairement, sans envoi à un serveur.

👉 Aucune donnée personnelle identifiante n’est traitée : l’extension se place hors du champ strict d’application du RGPD sur certains articles (ex. obligation de consentement explicite).

2. Transparence et finalité

  • ✅ Page de redirection qui informe l’utilisateur qu’il est sur le point d’utiliser un lien affilié.
  • ✅ Description claire dans le Chrome Web Store expliquant pourquoi le popup existe.

👉 Ces éléments satisfont aux obligations de transparence (Art. 12 à 14 RGPD) dès lors que le texte reste simple, compréhensible et sans jargon technique.

3. Consentement

  • ✅ Popup demandant le consentement explicite de l’utilisateur avant toute redirection ou modification d’URL.
  • ✅ Pas de traitement en arrière-plan sans accord : respect du principe du consentement libre, éclairé, spécifique et univoque (Art. 7 RGPD).

4. Pas de transfert de données à des tiers

  • ✅ Aucune donnée utilisateur n’est vendue, transférée ou exploitée par des tiers.
  • ✅ Aucun traqueur tiers ou outil de profilage n’est déployé.

5. Pas d’identification ni de profilage

  • ✅ L’extension ne suit pas les utilisateurs, ne collecte pas de logs d’usage ni d’historique personnel.

6. Stockage local uniquement

  • ✅ sessionStorage : données restent dans le navigateur et disparaissent à la fermeture de l’onglet.
  • ✅ Aucune base de données externe ou API distante n’est utilisée pour collecter ou stocker des données personnelles.

7. Sécurité

  • ✅ Code servi localement via chrome.runtime.getURL().
  • ✅ Aucune bibliothèque externe non fiable ni script distant n’est intégré.

➡️ Extension conforme aux exigences de minimisation des risques (Art. 32 RGPD).

Points important


• Notre politique de confidentialité est simple : « Nous ne collectons ni ne stockons aucune donnée personnelle. »
• Seul par votre conscentement, vous acceptez l’aspect affilié : « Si vous cliquez sur un lien partenaire, nous pourrions recevoir une commission sans frais supplémentaires pour vous. De plus, notre extension vous donne accès facilement au comparateur de prix google. Le comparateur de prix vous permettra de trouver le prix le moins chère tout en encouragant nos lien affiliés. »
